main module¶

main.abb_to_city(message)¶

Функция преобразования сокращений в название города

На вход получает сообщение, содержащее сокращённое название города, проверяет его наличие в словаре и, в случае успеха, возвращает название города.

Параметры

city_abbreviation – Сокращение города

Результат

Название города

Тип результата

string

main.callback_worker(call)¶

Обработчик ответов пользователей

На вход принимает ответ (подтверждение или отказ) пользователя в виде строк yes/no При положительном ответе информация заносится в базу данных, иначе выводит подсказку о начале работы с ботом.

Параметры

callback – Выбор yes/no пользователя

Результат

Информация о добавлении в базу данных или подсказка о начале работы с ботом

Тип результата

string

main.del_st(message)¶

Удаление прогноза из расписания (записи из БД)

Параметры

message (messageObject) – Пользовательское сообщение с текстом элемента расписания

Результат

Тип результата

bool

main.get_city(message)¶

Функция получения города и проверки его корректности

Принимает на вход строковую переменную с информацией о городе, после чего вызывает функцию установки времени set_time() или выводит сообщение о неверном вводе города.

Параметры

message (messageObject) – Пользовательское сообщение с названием города

Результат

Сообщение об успехе или ошибке получения города

Тип результата

string

main.get_city_id(city)¶

Функция получения идентификатора города

На вход получает сообщение с информацией о городе, после чего отправляет GET-запрос к OpenWeatherMap API, в результате получает идентификатор или сообщение о вызванном исключении, если не удалось получить JSON с информацией.

Параметры

city – Город

Результат

Идентификатор города

Тип результата

int

main.get_time(message)¶

Функция получения времени и проверки его корректности

Принимает на вход пользовательское сообщение с информацией о времени, после чего вызывает функцию подтверждения keyboard_confirmation() или выводит сообщение о неверном вводе времени.

Параметры

message (messageObject) – Пользовательское сообщение с информацией о времени

Результат

Сообщение со временем или сообщение об ошибке ввода времени

Тип результата

string

main.get_weather(city)¶

Функция получения погоды

На вход получает сообщение с информацией о городе, после чего отправляет GET-запрос к OpenWeaterMap API, в результате получает данные о погоде или сообщение о вызванном исключении, если не удалось получить JSON с информацией.

Параметры

city – Город

Результат

main.keyboard_confirmation(message)¶

Функция подтверждения добавления желаемого прогноза

Принимает на вход сообщение (информацию об отправителе сообщения), выводит две кнопки «Да» и «Нет», результат выбора которых передаётся в обработчик ответов.

Параметры

message (messageObject) – Пользовательское сообщение с информацией об отправителе

Результат

True, если вызов keyboard_confirmation() успешен, иначе False

Тип результата

bool

main.mail()¶

Функция отправления прогноза погоды

Непрерывно получает время, если оно совпадает со временем прогноза из базы данных, то получает прогноз погоды, а затем отправляет соответствующему пользователю.

Параметры

time – Текущее время

Результат

Сообщение пользователю с прогнозом погоды

Тип результата

messageObject
